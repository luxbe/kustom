## About

This project is designed to easily extract and work with klwp files

## Install

```sh
npm install kustom
```

## 💻 Develop

### 🛠️ Build with `wasm-pack build`

```
wasm-pack build --target web
```

### 🔬 Test with `cargo test`

```
cargo test
```

### 🎁 Publish to NPM with `wasm-pack publish`

```
wasm-pack publish
```
